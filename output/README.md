# Carpeta `output`

Artefactos derivados del procesamiento y análisis (generados por
`procesamiento/02-analisis.R`).

- `modelos.rds`: lista con los tres modelos MCO ajustados (M1–M3).
- `resultados.rds`: lista con los estadísticos y descriptivos que citan los capítulos.
- `figuras/`: gráficos en PNG (`fig-descriptivos.png`, `fig-h1.png`, `fig-h2.png`).
- `tablas/`: tablas exportadas (las tablas de regresión se construyen en los capítulos).

Estos archivos se regeneran ejecutando los scripts de `procesamiento/`.
